Secure Diploma Signing & Verification

ECDSA P-256 + SHA-256 — SEC3104 Advanced Cryptography Project

Step 1 — The university generates its ECDSA P-256 key pair. The private key signs diplomas and must be kept secret. The public key is shared with everyone for verification.

Step 2 — Enter the student's diploma details. The system combines content + metadata, computes a SHA-256 hash, and signs it with the university's private key.

Step 3 — Paste a diploma JSON package and verify it. The system recomputes the SHA-256 hash and checks the ECDSA signature against the university's public key.
