Secure Diploma System

SEC3104 Advanced Cryptography

University Key Generation

Mirrors keygenration.py — the university generates an ECDSA P-256 key pair. The private key is encrypted with a password and stored securely. The public key is distributed to all verifiers.

terminal
Issue & Sign Diploma

Mirrors sign.py — the university fills in student details, hashes the diploma content with SHA-256, signs it with the private key, and saves the package as diploma_{student_id}.json.

Available university keys: None yet — generate keys first in Tab 1.

Verify Diploma Authenticity

Mirrors verify.py — the verifier recomputes hash over the document field and checks the signature. No contact with the university is required.
